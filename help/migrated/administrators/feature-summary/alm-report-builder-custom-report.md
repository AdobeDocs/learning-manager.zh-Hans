# 在Report Builder中构建自定义报告

当您清楚了解所需的列和输出且现有模板与您的用例不匹配时，从头开始创建可获得最佳效果。 如果您是Report Builder新手，请考虑从模板开始。

在本例中，您将在每个经理下找到存在合规性课程风险的学习者。

1. 以管理员身份登录Adobe Learning Manager。
2. 选择&#x200B;**报表**，然后选择&#x200B;**Report Builder**。
3. 选择“**报表**”选项卡，然后选择“**创建报表**”。
4. 输入报告名称。需要名称。（可选）输入说明。
   ![](assets/image011.png)
5. 在“列”面板中，选择并展开以下数据集：
a. 用户
b. 学习对象
c. 用户符合性状态
6. 在要包含的以下列旁选择&#x200B;**+**。选定列将显示在报告画布中。
a.用户\名称
b.用户\经理姓名
c.学习对象\学习对象名称
d.用户符合性状态\完成%
e.用户符合性状态\符合性%
！  [](assets/image012.png)
7. 通过在画布中拖动列来重新排列顺序。
8. 要重命名列，请在列的别名字段中输入名称。 该别名显示为已下载文件中的列标题。
9. 选择&#x200B;**保存报告**。

## 下载报告

1. 选择右上角的&#x200B;**操作**。
   ![](assets/image013.png)
2. 选择下载。 报告准备好后，您可以通过通知图标进行下载。

以.csv文件扩展名下载的报告包含以下列：

1. name
2. managerName
3. name
4. completionPct
5. compliancePct

## 应用分组依据、筛选器和排序

### 过滤器

现在您已下载报告，请应用completionPct OR compliancePct等于100的过滤器。

1. 打开报告并选择右上角的&#x200B;**编辑**。
2. 选择&#x200B;**添加筛选器**并搜索要应用筛选器的列。
   ![](assets/image014.png)
3. 选择&#x200B;**添加**。
4. 将筛选器与AND/OR逻辑组合；选择在筛选器行之间切换运算符。
   ![](assets/image015.png)
5. 选择&#x200B;**保存报告**&#x200B;并下载报告。

下载的报告包含completionPct OR compliancePct等于100的记录。

### 分组依据

按经理将记录分组到：

* 按经理聚合学习者数据
* 计算管理器级别平均值
* 按每个经理对学习者进行计数

1. 打开报告并选择右上角的&#x200B;**编辑**。
2. 选择&#x200B;**“分组依据”:Select**，然后选择&#x200B;**“用户管理器名称”**列。
   ![](assets/image016.png)
3. 聚合以下列：
a. 用户\名称
b. 学习对象\学习对象名称
4. 选择&#x200B;**Count**作为列的聚合函数。
   ![](assets/image017.png)
5. 对“学习对象”\“学习对象名称”重复此操作。
   ![](assets/image018.png)
6. 选择&#x200B;**保存报告**&#x200B;并下载报告。

下载的报告包含经理对学习者培训绩效的摘要。 它显示了每个经理的平均完成率、平均相容性分数和总学习者计数。 数据显示，所有小组都完成了通用培训，而管理人员之间的合规性绩效差异很大。

### 排序

按每个经理的学习者人数对报告进行降序排序。

1. 打开报告并选择右上角的&#x200B;**编辑**。
2. 选择&#x200B;**添加排序**。
3. 搜索用户名并选择&#x200B;**用户\名称**。
4. 选择&#x200B;**降序**。
   ![](assets/image019.png)
5. 选择&#x200B;**添加**。
6. 选择&#x200B;**保存报告**&#x200B;并下载报告。

下载的报告包含每位经理的学习者数量，并以降序排列。
