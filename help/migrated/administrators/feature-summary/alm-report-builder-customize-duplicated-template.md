# 自定义重复的Report Builder模板

复制模板后，您可以添加或删除列、更新过滤器、重命名列以及调整排序，然后再保存和下载\。 本文介绍了每个自定义步骤。

在开始之前，请从&#x200B;**模板**&#x200B;选项卡复制模板。

## 添加和删除列

1. 在编辑视图中，找到包含要添加的列的数据集。
2. 选择列名称旁边的&#x200B;**+**。 该列将显示在报告画布中。
3. 多次添加同一列，例如，对两个不同的状态值进行计数。 再次为该列选择&#x200B;**+**。
4. 要删除列，请在画布中选择它，然后选择删除图标。

## 重新排序和重命名列

1. 将列拖到画布中的所需位置。 画布中的顺序与下载文件中的列顺序匹配。
2. 要重命名列，请选择其别名字段，然后输入要作为列标题在报告中显示的名称。

## 更新过滤器

1. 要更改现有筛选器值，请选择筛选器行，编辑值，然后确认。
2. 要添加新筛选器，请选择&#x200B;**添加筛选器**。
3. 选择要过滤的字段。
4. 选择一个运算符。 可用的运算符取决于字段的数据类型。
5. 输入或选择一个值：
   * 对于字符串字段，请键入以搜索并从建议中进行选择。
   * 对于列表字段，请从选取器中选择一个或多个值。
   * 在日期字段中，输入日期或选择相对范围，例如最近90天。
6. 要删除筛选器，请选择该筛选器行上的“删除”图标。

## 更新排序

1. 在要作为排序依据的列上选择排序图标。
2. 选择&#x200B;**升序**&#x200B;或&#x200B;**降序**。
3. 要对其他列进行排序，请对每一列重复此操作。 第一个排序的列为主列；其他排序在领带中应用。

>[!TIP]
>
>始终至少应用一种排序。 如果不排序，同一报告的下载之间的行顺序可能会不同。

## 保存和下载

1. 选择保存以保存您对&#x200B;**报告**&#x200B;选项卡所做的更改。
2. 选择&#x200B;**下载**&#x200B;以生成报告。 当文件准备就绪时，您将收到应用程序内通知。
