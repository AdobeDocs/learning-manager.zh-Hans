# 下载、共享和订阅报告

在Report Builder中保存报告后，您可以按需下载该报告、与帐户中的其他管理员共享该报告，或订阅以定期接收该报告。

## 下载报告

1. 在&#x200B;**报告**&#x200B;选项卡中，找到要下载的报告。
2. 选择&#x200B;**下载**。
3. 选择&#x200B;**确定**。
   ![](assets/image046.png)
报告生成以异步方式运行。 当文件准备就绪时，您将收到应用程序内通知。
4. 打开通知并下载文件。

>[!NOTE]
>
>例如，如果报告没有匹配的数据，则由于筛选器未返回任何结果，因此仍然会生成空文件。 您不会收到错误；下载的文件将有标头，但没有行。

## 在Report Builder中共享和订阅报告

向其他管理员授予对已保存报告的访问权限，并在Report Builder中设置计划的电子邮件发送。

**共享和订阅**&#x200B;对话框包含两个独立的部分：一个用于与其他管理员共享报告，另一个用于设置电子邮件订阅。 您可以使用其中之一或两者。

## 与其他管理员共享报告

共享管理员可以查看、复制和编辑报告。

1. 打开要共享的报告。
2. 选择&#x200B;**操作** > **共享和订阅**。
   ![](assets/image_047.png)
3. 在&#x200B;**具有共享访问权限的管理员**&#x200B;下，选择&#x200B;**编辑**。 然后选择&#x200B;**选择用户/用户组**&#x200B;字段。
4. 搜索并选择要与之共享的管理员或用户组。
   ![](assets/image048.png)
5. 选择&#x200B;**“保存”**。

选定管理员现在可以在其Report Builder视图中访问报告。

>[!NOTE]
>
>具有共享访问权限的管理员可以查看、复制和编辑报告。 要删除访问权限，请返回&#x200B;**共享和订阅**，并取消选择用户或用户组。

## 为管理员订阅报告

订阅管理员会根据您选择的频率通过电子邮件收到报告。

1. 打开要向管理员订阅的报告。
2. 选择&#x200B;**操作** > **共享和订阅**。
3. 在&#x200B;**订阅管理员**&#x200B;下，选择&#x200B;**编辑**。
4. 选择&#x200B;**电子邮件频率**&#x200B;下拉菜单。
5. 选择频率：
   * **每日发送**
   * **每周发送**
   * **每月发送**
6. 在&#x200B;**选择用户/用户组**&#x200B;字段中，搜索并选择要订阅的管理员或用户组。
7. 选择&#x200B;**“保存”**。

已订阅的管理员会按所选频率通过电子邮件接收报告。

>[!TIP]
>
>在设置订阅之前，至少对报表应用一种排序。 这可确保每个计划交付的行顺序一致。
