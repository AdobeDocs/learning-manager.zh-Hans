# Report Builder — 常见问题解答


**模板和报告之间有何区别？**

模板是Adobe Learning Manager提供的预构建报告配置。 它们专为常见用例设计，可立即下载，且为只读状态。 您无法编辑它们。 报告是您自己保存的配置。 您可以通过从头开始构建来创建模板，也可以通过复制模板和编辑副本来创建模板。 报告显示在您的&#x200B;**报告**&#x200B;选项卡中；模板显示在&#x200B;**模板**&#x200B;选项卡中。

**我可以直接编辑模板吗？**

数字 模板是只读的。 要自定义模板，请选择&#x200B;**复制**&#x200B;以创建可编辑副本。 您所做的更改将另存为新报告到&#x200B;**报告**&#x200B;选项卡中，且不会影响原始模板。

当数据中的字段与主记录具有一对多关系时，会出现重复行。 常见原因包括多名讲师的会话（每位讲师每个会话一行）以及多名作者的学习对象。 要解决此问题，请将具有多个值（例如&#x200B;**讲师姓名**&#x200B;或&#x200B;**作者姓名**）的字段添加为列。

**为什么在同一报告的两个下载之间更改行顺序？**

报告生成使用分布式查询系统。 如果不进行显式排序，行顺序取决于哪个查询节点首先返回结果，这在不同的运行之间可能会有所不同。 对报告至少应用一种排序，以确保在下载过程中输出的一致性。

**为什么日期以UTC显示？**

在此版本中，Report Builder以UTC返回日期值。 在将来的版本中，您的帐户配置的时区将应用于日期字段。 分析基于日期的数据时，UTC偏移中的因子与帐户的主要时区相关。

**新注册或完成数据显示需要多长时间？**

Report Builder从Adobe Learning Manager的数据库提取，从系统中发生事件算起，该数据库的最大延迟大约为15分钟。 如果您刚刚录制了注册或完成，但报告中未显示，请等待至少15分钟，然后再次下载。

**报表中的行或列是否有限制？**

报告限制为大约100万行。 此版本中的列数没有限制。 如果您的报告需要100万行以上，请应用过滤器以缩小范围。

**我是否可以通过API或自动化提取Report Builder数据？**

计划在未来版本中自动以API访问Report Builder报告。 在当前版本中，可通过Report BuilderUI手动下载报告，也可通过订阅功能按计划接收报告。
