# 徽章

徽章是一种成就度量，员工在完成课程后即可获得。 Adobe Learning Manager推出了名为“徽章”的电子学习概念。 全球各地的专业人士可将这些徽章用于表现其掌握了特定技能或学习成就。

您可以对徽章进行定义，以此激励用户。

管理员可以为学习者创建徽章，如下所示：

1. 以管理员身份登录Adobe Learning Manager 。
2. 在左侧窗格中选择&#x200B;**[!UICONTROL 徽章]**。 此时会显示学习者的徽章列表。

   >[!NOTE]
   >
   >默认情况下，我们会提供一份示例徽章的列表。

3. 在页面右上角选择“**[!UICONTROL 添加]**”。 此时会显示&#x200B;**[!UICONTROL 添加徽章]**&#x200B;对话框。

   ![](assets/add-badge1.png)

   *添加徽章名称及其图像*

4. 键入&#x200B;**[!UICONTROL 徽章名称]**。 要上传徽章，请单击&#x200B;**[!UICONTROL “上传徽章”]**，然后单击&#x200B;**[!UICONTROL “保存”]**。

>[!NOTE]
>
>若要在学习者完成课程或学习路径后添加自定义徽章或证书，请通过[learningmanagersupport@adobe.com](mailto:learningmanagersupport@adobe.com)联系Adobe Learning Manager支持团队。
