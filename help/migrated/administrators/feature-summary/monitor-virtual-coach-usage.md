# 监视虚拟引导使用情况

在Adobe Learning Manager中查看虚拟教程使用数据、跟踪每月活动用户(MAU)积分消费以及访问学习者绩效报告。

## 为您的帐户激活虚拟辅导

虚拟引导可用作Adobe Learning Manager的加载项。 购买后，配置会生成一个激活密钥，并将此密钥通过电子邮件发送给帐户管理员。

1. 以管理员身份登录Adobe Learning Manager 。
2. 从左侧导航窗格中导航到&#x200B;**帐单**&#x200B;页面。
3. 在&#x200B;**虚拟教程**&#x200B;部分中，输入通过电子邮件收到的激活密钥。
4. 选择&#x200B;**应用**。已为您的帐户启用虚拟辅导。
   ![](assets/virtual-coach-037.png)

激活后，您将收到应用程序内通知，确认该功能已启用。 四个示例角色扮演场景会自动添加到内容库，以便作者可以立即开始。

>[!NOTE]
>
>激活密钥在设置过程中自动生成，并通过电子邮件共享。 如果您没有激活密钥，请联系您的Adobe Learning Manager客户成功经理(CSM)。

## 查看您的MAU信用余额

每月活动用户(MAU)积分统计每月使用虚拟教程的唯一学习者数。

1. 导航至&#x200B;**帐单**&#x200B;页面。
2. 在&#x200B;**虚拟辅导**&#x200B;部分中，选择&#x200B;**查看使用情况详细信息**。
3. 使用&#x200B;**选择时段**&#x200B;下拉列表选择要查看的日期范围。
   ![](assets/virtual-coach-038.png)

   **整体使用情况**&#x200B;表显示：

   a. **可用：**&#x200B;已购买的MAU积分总数\
   b. 已使用&#x200B;**：至今已使用**&#x200B;个积分\
   c. **剩余积分：**&#x200B;积分可用于剩余的合同期

   **每月使用情况**&#x200B;表按日历月显示唯一活跃学习者数。

4. 选择&#x200B;**下载详细报告**&#x200B;以导出完整使用情况数据。

## MAU积分的使用方式

学习者在日历月完成虚拟引导会话时，系统会消耗MAU积分。 同一学习者在同月完成的其他会话不会使用其他积分。

| 情景 | 消耗的MAU |
|----------|--------------|
| 一名学习者在1月完成了5个课程 | 1 |
| 同一学习者在1月和2月使用虚拟辅导 | 2（每月1个） |
| 100名学习者各自在1月完成1个课程 | 100 |

每个日历月会为每个唯一学习者计算&#x200B;_MAU积分，无论每个学习者完成多少会话。_

于合约期末尚未动用之信贷失效，且并无结转。

### 示例1：单个学习者、多个会话

情景：Sarah在1月完成5次虚拟指导会话。

* 1月届会：5次
* 消耗的MAU：1
* 原因：Sarah被计为1月份的单个唯一用户，无论她练习了多少次。

### 示例2：同一学习者，多个月

情景：Sarah在1月和2月都使用虚拟教练。

* 1月届会：3次
* 2月届会：2次
* 消耗的MAU：2个（1月1个+ 2月1个）
* 原因：每个日历月单独计算。

### 示例3：多个学习者，同月

情景：1月份，100名销售代表每完成一次虚拟引导课程。

* 课程总数：100
* 消耗的MAU：100
* 原因：每个单独的学习者均计为当月一个MAU。

### 示例4：不同时间的团队练习

情景：您的50人团队全年都使用虚拟教练。

| 月 | 活跃学习者 | 本月消耗的MAU | 累计MAU |
|------|----------------|--------------------------|-----------------|
| 每年 | 0 | 0 | 0 |
| ebruary | 5 （5人未进行练习） | 5 | 5 |
| 三月 | 0（50个练习增益） | 0 | 45 |
| 四月 | 0 | 0 | 75 |

## 查看虚拟引导报告

**管理**&#x200B;部分> **报告**&#x200B;部分> **AI报告**&#x200B;页面提供整个组织内所有虚拟引导程序活动的使用情况和性能数据。 在&#x200B;**虚拟教程**&#x200B;标题下提供了两个报告。

所有报告均以CSV格式导出。 生成报告可能需要几分钟时间，具体取决于数据大小。

### 学习者使用情况摘要

包含所有学习者的每月使用情况数据。 使用此报告可以跟踪每月使用虚拟教程的学习者数量，监控MAU积分使用情况，并识别一段时间内的参与趋势。

### 会话详细信息

包含过去90天内所有学习者的会话级别数据。 使用此报告可以检查整个学习者群体的单个会话分数、主题涵盖范围和样式指标，并确定可能需要额外培训或内容的技能差距。

## 访问和下载报告

1. 以管理员身份登录Adobe Learning Manager 。
2. 在左侧导览窗格中选择&#x200B;**报表**。
3. 选择&#x200B;**AI报表**。
4. 在&#x200B;**虚拟辅导**&#x200B;部分下，选择要下载的报告、**学习者使用情况摘要**&#x200B;或&#x200B;**会话详细信息**。
   ![](assets/virtual-coach-039.png)
5. 出现提示时选择日期范围，然后选择&#x200B;**继续**。
6. 报告会自动下载为CSV文件。
