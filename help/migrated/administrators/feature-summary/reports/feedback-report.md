# 反馈报告

## 概述

学习者完成学习对象后，Adobe Learning Manager中的反馈报告会同时收集1级（学习者反馈）和3级（经理反馈）。 该报告对学习者及其经理的主观和客观回答进行了结构化概述。

该报告跟踪学习者详细信息（如姓名、电子邮件、反馈表单名称、版本和语言），并捕获对管理员定义问题的响应。 还记录学习者对每个问题的选择（例如，强烈同意、同意或不同意）。

## 目的和好处

* 提供可操作的洞察信息，以提高课程质量和整体学习效果。
* 根据直接用户反馈优化导航、步调和交付方法。

## 用例

* **快速确定内容问题**：管理员可以发现评分低或反复出现负面评论，并更新学习对象，而无需等待支持票证或升级。
* **衡量培训效果**：团队可以比较多个课程或版本中的学习者反馈，以了解哪些学习对象表现良好，哪些可能需要重新工作。
* **使用反馈表单跟踪学习者参与情况**：管理员可以查看有多少学习者正在回答或跳过问题，这有助于他们改进反馈表单以提高响应质量和完成率。

## 如何下载反馈报告

1. 以管理员身份登录Adobe Learning Manager。
2. 从左侧导航菜单中选择&#x200B;**[!UICONTROL 报表]**。
   ![](assets/select-report.png)
   _管理员主页显示了用于下载报告的“报告”选项（突出显示）_

3. 在报表中选择&#x200B;**[!UICONTROL 自定义报表]**，然后选择&#x200B;**[!UICONTROL Excel报表]**。
4. 选择&#x200B;**[!UICONTROL 反馈报告]**。
   ![](assets/select-feedback-report.png)
   _自定义报告部分显示选择“反馈报告”以访问学习者和经理反馈数据的选项_

5. 选择&#x200B;**[!UICONTROL 所有培训]**&#x200B;或&#x200B;**[!UICONTROL 所选培训]**以及日期范围。 如果选择第二个选项，您最多可以添加10个课程或学习路径并生成反馈报告。 此外，您最多可以生成一年的报告。
   ![](assets/feedback-report.png)
   _通过选择培训范围、设置日期范围并在下载之前选择翻译选项来配置反馈报告_

6. 选择要将L1反馈翻译成的语言。 当明确定义目标问题及其回答语言版本时，该语言版本将被翻译成选定的语言。 报告中只会显示以选定语言明确定义的主观问题。  对主观问题的答复将采用原始答复语言。
7. 选择&#x200B;**[!UICONTROL 下载]**&#x200B;以下载报告。

## 反馈报告包含哪些内容

以下是帐户级别报告中的默认列：

| 列 | 描述 |
|----|----|
| 反馈类型 | 指示反馈来自学习者(L1)还是经理(L3) |
| 用户名 | 完成培训的学习者姓名 |
| 用户电子邮件 | 学习者的电子邮件地址 |
| 培训 ID | 系统生成的分配给每个学习对象（课程、认证或学习路径）的唯一标识符 |
| 培训名称 | 为其提交反馈的学习项目的名称 |
| 培训实例 | 培训实例名称（对于多实例课程） |
| 培训类型 | 培训类型（课程、认证、学习路径） |
| 反馈表单类型 | 使用的反馈表单的类型/类别（例如，混合课程、自学课程和教室课程） |
| 反馈表单名称 | 反馈表单的名称 |
| 反馈版本 | 反馈表单的版本号 |
| 当前经理 | 学习者经理的姓名 |
| 当前经理电子邮件 | 学习者经理的电子邮件地址 |
| 完成日期 | 学习者完成培训的日期 |
| 反馈日期 | 学习者提交反馈的日期 |
| L1反馈原始语言 | 学习者最初提交L1反馈时使用的语言 |
| L3李克特量表问题1 | 使用评分表测量学习者的培训后表现 |
| L3李克特比例响应1 | 经理对李克特级问题的回答 |
| L3自由文本问题1 | 在L3反馈表单中为经理添加了自由文本问题；可配置为可选或必填 |
| L3自由文本响应1 | 经理对这个自由文本问题的回答 |

根据添加到反馈表单中的四类问题，帐户级别报告中将显示以下列：

| 列 | 描述 |
|---|---|
| 课程效果 | 表单中显示的预定义课程效果问题 |
| 课程效果答案 | 学习者对课程效果问题的回答 |
| NPS问题1 | 第一个网络发起人得分问题 |
| NPS范围1 | 使用的评级表（例如，0至10） |
| NPS答案1 | 关于NPS问题1的学习者评分 |
| 李克特问题1 | 第一个李克特级问题 |
| 李克特答案1 | 对李克特问题1的答复 |
| 文本问题1 | 表单中的第一个开放式问题/文本问题 |
| 文本答案1 | 学习者对文本问题1的响应 |


>[!NOTE]
>
>帐户级别报告还包括为学习者配置的所有活动字段。

“学习对象”级别报告中显示以下列：

| 列 | 描述 |
|---|---|
| 学习者无法注册此类课程。 | 学习者姓名 |
| 电子邮件 | 学习者的电子邮件地址 |
| 反馈表单名称 | 反馈表单的名称 |
| 反馈版本 | 反馈表单的版本号 |
| 学习者语言 | 学习者选择的语言 |

>[!NOTE]
>
>“学习对象”级别报告还包含已添加到反馈表单的问题。 每个问题都显示为列标题，学习者对这些问题的回答显示在下面的相应行中。
