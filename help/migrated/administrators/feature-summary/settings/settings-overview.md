# 设置

## 概述

Adobe Learning Manager提供了详细的管理员设置，可让您配置和自定义学习管理系统以满足组织的特定需求。 这些设置被分为几个关键类别，每个类别提供不同的管理功能。

### 基本设置

* 此部分允许管理员配置基本组织详细信息，包括国家/地区、时区或区域设置。
* 为学习对象配置反馈机制和评级系统。
* 影响整体平台行为和使用情况的其他常规设置。

有关详细信息，请参阅Adobe Learning Manager中的[设置](/help/migrated/administrators/feature-summary/settings/basic-settings.md)。

### 集成设置

* 配置身份验证选项，包括Adobe ID和单点登录(SSO)。
* 管理与外部数据系统的集成，以实现用户配置、课程目录和报告数据同步。
* 设置和管理与其他Learning Manager帐户的连接，以实现内容共享和协作学习计划。

有关更多信息，请参阅Adobe Learning Manager中的设置。

### 高级设置

* 创建和管理自定义标签以组织和分类学习内容，方便学习者查找相关材料。
* 针对讲师主导的培训课程设置和管理物理教室位置，包括容量、设备和日程安排详细信息。
* 配置合规性和组成功信息板。

有关更多信息，请参阅Adobe Learning Manager中的设置。
