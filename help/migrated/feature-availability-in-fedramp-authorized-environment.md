# FedRAMP授权环境中的功能可用性

## 简介

作为FedRAMP安全和合规性要求的一部分，FedRAMP授权的环境中目前不支持Adobe Learning Manager的以下功能。 实施这些限制是为了保护受监管的数据，维护边界控制，以及维护规定的审计标准。 我们正在积极跟踪这些约束，并将随着FedRAMP态势的演变，通报可用性方面的任何未来变化。

| [管理员AI助手](/help/migrated/administrators/feature-summary/alm-ai-assistant.md) | 不支持 |
| [学习者AI助手](/help/migrated/learners/feature-summary/learner-ai-assistant.md) | 不支持 |
| [AI支持的搜索](/help/migrated/learners/feature-summary/advanced-search.md) | 不支持 |

| [改进的Recommendations](/help/migrated/recommendations-adobe-learning-manager.md) | 不支持 |
