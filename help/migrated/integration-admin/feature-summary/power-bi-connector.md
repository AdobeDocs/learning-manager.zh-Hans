# Adobe Learning Manager中的Power BI连接器

## 简介

通过Power BI连接器，可将Adobe Learning Manager与MicrosoftPower BI（商业许可证）集成，以便分析、可视化和共享学习数据。

通过此集成，集成管理员可以自动将实时数据集（如学习者成绩单、用户技能和xAPI活动报告）直接导出到选定的Power BI工作区。

连接后，您可以使用Power BI的完整功能来创建自定义信息板和报表。 这有助于您的组织更深入地了解学习者进度、技能成就和培训效果，并根据实时学习数据做出明智决策。

>[!NOTE]
>
>Adobe Learning Manager仅支持与MicrosoftPower BI的商业版本集成。 不支持与政府云版本集成。

## 先决条件

- 仅支持具有&#x200B;**商业许可证**&#x200B;的Power BI。
- 确保您有权创建Power BI应用程序和工作区。
- 获取您的&#x200B;**租户名称**、**应用程序客户端ID**、**应用程序客户端密钥**&#x200B;和&#x200B;**工作区ID**（可选）。

## 配置Power BI连接器

要将ALM与Power BI连接：

1. 以集成管理员身份登录Adobe Learning Manager.
2. 将鼠标悬停在&#x200B;**Power BI**&#x200B;连接器图块上，然后选择&#x200B;**连接**。

   ![](assets/power-bi-connector1.png)
   _选择“连接”以配置Power BI连接器_

3. 键入以下详细信息：

   - 客户端 ID
   - 客户端密钥
   - 租户名称
   - 工作区Id（可选）

   ![](assets/power-bi-connector2.png)
   _键入所需的详细信息以配置Power BI_

4. 选择&#x200B;**连接**。

## 注册Power BI应用程序

要注册Power BI应用程序，请执行以下操作：

1. 转到[注册Power BI应用](https://app.powerbi.com/embedsetup)。
2. 为您的组织选择&#x200B;**嵌入**，然后登录到您的Microsoft帐户。
3. 键入应用程序的名称。
4. 在&#x200B;**应用程序类型**&#x200B;下选择&#x200B;**服务器端Web应用程序**。
5. 在&#x200B;**重定向URL**&#x200B;部分中，选择&#x200B;**使用自定义URL**，然后键入[此URL](https://learningmanager.adobe.com/ctr/app/azure/_callback)： （根据您的环境需要，替换域。）
6. 在&#x200B;**主页URL**&#x200B;字段中，键入[此URL](https://learningmanager.adobe.com/)。
7. 在&#x200B;**权限**&#x200B;部分中，选择&#x200B;**读取所有数据集**&#x200B;和&#x200B;**读取和写入所有数据集**。
8. 联系您的Power BI管理员以获取&#x200B;**租户名称**。
9. 如果您没有Workspace ID，请在Power BI中创建工作区（需要Power BIPro）并从URL复制ID。
10. 选择&#x200B;**注册应用**&#x200B;并保存&#x200B;**客户端ID**&#x200B;和&#x200B;**客户端密钥**&#x200B;供以后使用。

>[!NOTE]
>
>如果以后需要重新授权连接，请创建一个新的Power BI应用程序并使用适用于您的环境的正确重定向URL。

## 将报告导出到Power BI

配置连接后，您可以导出以下报告：

- **学习者成绩单**
- **用户技能**
- **xAPI活动报告**
- **统一报告**（多个报告的组合）

### 学习者成绩单

#### 计划导出

1. 在左侧面板中选择&#x200B;**学习者成绩单**。
2. 在“导出”页面上选择&#x200B;**启用计划**。
3. 选择&#x200B;**开始日期**&#x200B;和&#x200B;**时间**。
4. 定义&#x200B;**间隔**&#x200B;以确定导出重复频率（每天、每周等）。

   ![](assets/power-bi-connector3.png)
   _为学习者成绩单启用计划导出_

5. 选择&#x200B;**“保存”**。

#### 按需导出

- 您可以通过指定&#x200B;**开始日期**&#x200B;并运行按需导出来手动生成报告。
- 该报告将包括从指定日期到现在的数据。

### 用户技能

#### 计划导出

1. 在左侧面板中选择&#x200B;**用户技能**。
2. 在“导出”页面上选择&#x200B;**启用计划**。
3. 选择&#x200B;**开始日期**&#x200B;和&#x200B;**时间**。
4. 定义&#x200B;**间隔**&#x200B;以确定导出重复频率（每天、每周等）。

   ![](assets/power-bi-connector4.png)
   _启用计划导出用户技能报告_

5. 选择&#x200B;**“保存”**。

#### 按需导出

- 您可以通过指定&#x200B;**开始日期**&#x200B;并运行按需导出来手动生成报告。
- 该报告将包括从指定日期到现在的数据。

### 管理xAPI活动报告

**xAPI语句**&#x200B;也可以导出到Power BI。

#### 配置xAPI导出

1. 选择&#x200B;**导出xAPI活动报告**。
2. 在左侧窗格中选择&#x200B;**配置**。

   - 填写JSON路径字段以匹配CSV列。
   - 选择“**添加**”以包含更多路径。
   - 使用&#x200B;**编辑**&#x200B;更新字段。
3. 选择&#x200B;**“保存”**。

#### 计划导出

1. 选择&#x200B;**配置计划**。
2. 选择&#x200B;**启用使用此连接导出xAPI语句**。
3. 设置&#x200B;**开始日期**、**时间**&#x200B;和&#x200B;**间隔**。
4. 选择&#x200B;**“保存”**。

#### 按需导出

1. 选择&#x200B;**按需**。
2. 指定&#x200B;**开始日期**。
3. 选择&#x200B;**执行**。

>[!NOTE]
>
>如果学习记录存储区(LRS)中的某些xAPI语句缺少已配置的JSON路径，则其值将在Power BI中显示为N/A。

#### 查看执行状态

- 使用&#x200B;**执行状态**&#x200B;查看导出的历史记录，包括开始时间、持续时间和状态。
- 警告图标表示运行失败。 单击此链接可将错误报告下载为.CSV。

### 统一报告

**统一报表**&#x200B;合并来自以下各项的数据：

- 学习者成绩单
- 游戏
- 反馈报告
- 登录/访问
- 用户技能
- 用户报告
- 培训报告

这有助于您通过在Power BI中合并数据来创建更强大的面板。

#### 创建统一报告配置

1. 选择&#x200B;**统一报告**，然后选择&#x200B;**配置**。
2. 在&#x200B;**数据集名称**&#x200B;字段中键入唯一的名称。
3. 在&#x200B;**选择用于数据导出的报告**&#x200B;下，选择一个或多个要包含在此数据集中的报告。

   - 学习者成绩单
   - 登录/访问
   - 培训报告
   - 游戏
   - 用户技能
   - 反馈报告
   - 用户报告
4. 使用&#x200B;**添加用户组过滤器**&#x200B;字段选择要导出的用户组数据。 默认情况下，**所有用户**&#x200B;处于选中状态。
5. 使用&#x200B;**添加内容目录筛选器**&#x200B;字段可按内容目录筛选报告。
6. 筛选器表显示哪些报表支持&#x200B;**用户组**、**目录**&#x200B;或&#x200B;**时间**&#x200B;筛选器。

   ![](assets/power-bi-connector5.png)
   _为统一报告创建配置_

7. 选择报表和筛选器后，选择右上角的&#x200B;**保存**。

#### 按状态筛选学习者成绩单

- **全部：**&#x200B;日期范围内的所有记录
- **已完成：**&#x200B;仅已完成的学习活动
- **正在进行：**&#x200B;仅正在进行的活动
- **未启动：**&#x200B;排除尚未启动的记录
- **取消注册：**&#x200B;包括取消注册的记录

## 下载Power BI模板

Adobe提供现成的Power BI模板，帮助您快速入门。

- 下载模板、导入报告并根据需要进行自定义。
- 使用模板无需从头开始即可创建引人注目的仪表板。

## 学习路径相关设置

**学习路径**&#x200B;如何在您的报告中显示取决于您的管理员设置：

- **现有连接：**

   - 如果禁用了&#x200B;**学习路径**，则不包括任何相关的行或列。
   - 如果启用，报告则包含注册学习者的学习路径（较高级别）。

- **新连接：**

   - 如果禁用了“学习路径”，则各列会显示：

      - **嵌入式路径：**&#x200B;学习计划名称。
      - 学习计划的&#x200B;**嵌入式路径ID：** ID。
      - **嵌入式课程ID：**&#x200B;学习路径中的课程ID。
   - 如果启用，**类型**&#x200B;列将在相关位置使用学习路径（更高级别）。
   - 对于新连接，更改将在30天后应用。

### 在哪里查看您的数据**

所有导出的数据都显示为您Power BI帐户下的数据集。 使用这些控件可以构建自定义功能板和可视化效果。
