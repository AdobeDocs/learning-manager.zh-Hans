# 解决Salesforce (SFDC)与Adobe Learning Manager的集成问题

## 解决SFDC导出失败问题（2-3个小时以上没有导出）

如果SFDC导出未成功完成&#x200B;**2-3小时**&#x200B;以上，请使用以下步骤识别并了解问题。

1. 转到&#x200B;**Salesforce主页**。
2. 在&#x200B;**“快速查找”**&#x200B;搜索框中，键入&#x200B;**`Bulk Data Load Jobs`**&#x200B;并打开它。
3. 该页面显示&#x200B;**过去7天**&#x200B;的批量数据加载作业。
4. 查找与&#x200B;**Adobe Learning Manager (ALM)对象**&#x200B;相关的作业。
5. 单击要调查的特定作业。
6. 滚动到作业详细信息页面的&#x200B;**底部**&#x200B;以查看&#x200B;**错误消息**&#x200B;和其他详细信息。

在许多情况下，根本原因是SFDC **中的字段级别权限不足**。 失败的字段名称可能如下所示：

- `cp_Module_ID`
- 或其他与ALM自定义对象相关的`cp_...`字段。

如果错误中显示此类字段，请继续执行下一部分。

## 向SFDC中的ALM自定义对象字段授予字段权限

使用&#x200B;**字段辅助功能**&#x200B;功能可修复特定字段（例如，`cp_Module_ID`）的权限问题。

### 步骤

1. 转到&#x200B;**Salesforce主页**。
2. 在&#x200B;**“快速查找”**&#x200B;搜索框中，键入&#x200B;**`Field Accessibility`**&#x200B;并打开它。
3. 从对象列表中，选择相关的&#x200B;**报表类型/对象**。
   - 示例：学习者成绩单(LT)报告的`cp_LearnerTranscript_xxxx_xxxx`。
4. 在下一页上，选择&#x200B;**“按字段查看”**。
5. 在&#x200B;**字段下拉列表**&#x200B;中，选择导出错误中显示的字段
   - 示例： `cp_Module_ID`。
6. 在访问矩阵中，单击&#x200B;**系统管理员**&#x200B;配置文件的&#x200B;**“隐藏”**&#x200B;条目（如果需要，单击其他相关配置文件）。
7. 在下一页上：
   - 在适当的位置启用&#x200B;**“可见”**。
   - 单击页面底部的&#x200B;**保存**。
8. 保存后，可返回到字段辅助功能视图。 对于&#x200B;**系统管理员**（以及您更新的任何其他配置文件），该字段现在应显示为&#x200B;**“可编辑”**。

## 对所有受影响的字段重复此操作，然后重试导出

1. **重复** **每个字段**&#x200B;的第2节中的字段辅助功能步骤报告在&#x200B;**批量数据加载作业**&#x200B;中存在权限问题。
2. 一旦所有有问题的字段都具有适当的可见性和编辑权限：
   - 返回&#x200B;**Adobe Learning Manager**。
   - 在&#x200B;**SFDC连接器**&#x200B;配置中，**重试导出**。


## 重要说明和限制

在设计或排除集成故障时，请记住这些SFDC-ALM细节。

### SFDC中没有自动创建对象/字段

- **SFDC连接器未在Salesforce**&#x200B;中创建新对象或字段。
- 如果在ALM **中添加了**&#x200B;新字段，并且您希望该字段显示在SFDC中：
   - 在SFDC中手动&#x200B;**创建相应的自定义字段**。
   - **将** SFDC自定义字段映射到连接器配置中的&#x200B;**适当的ALM字段**。
   - 确保新字段具有&#x200B;**适当的字段级别权限**（使用第2部分）。

### 具有自定义域的ALM帐户的回调URL

- 如果ALM帐户使用&#x200B;**自定义域**，则工程团队必须&#x200B;**将该自定义域**&#x200B;添加到SFDC连接器的OAuth生产应用程序的&#x200B;**回调URL允许列表**&#x200B;中。
- 此操作是通过&#x200B;**Jira请求**&#x200B;发送给工程部门的。
- 否则，连接器的OAuth流可能会失败。

### 时区限制（仅限UTC）

- 如果SFDC组织使用&#x200B;**非UTC时区**，则在同步期间可能会丢失&#x200B;**完成和注册数据**。
- 原因： ALM **当前仅支持UTC时区**&#x200B;进行SFDC集成。
- 内部参考：引发PAPI-24525以支持更多时区。

### 筛选器数据类型限制（选择列表与清单）

- ALM支持导入使用选择列表字段&#x200B;**创建的** SFDC筛选器。
- ALM **不支持**&#x200B;基于清单/多选复选框字段的筛选器。
- 原因： ALM **仅支持SFDC筛选器的字符串数据类型**。
