# 学习者主页

## 概述 {#overview}

管理员启用沉浸式布局后，学习者在登录应用程序时会看到一个经过全面改进的用户界面。

>[!NOTE]
>
>IE11 浏览器不支持沉浸式布局。

## 沉浸式布局的新学习者 UI

>[!IMPORTANT]
>
>我们将分阶段发布新的学习者 UI。

我们对学习者 UI 进行了更新，使其更加时尚和更新。 新 UI 旨在跨学习者主页&#x200B;]**、我的学习]**、**[!UICONTROL 目录]**&#x200B;和&#x200B;**[!UICONTROL 课程概述]**&#x200B;登录页面提供&#x200B;**[!UICONTROL 一致的**[!UICONTROL &#x200B;用户体验。新的视觉元素遵循当前的设计风格，使产品看起来更易于使用和更具吸引力。 此更新包括新的刊头、侧面板和现代小部件。

>[!NOTE]
>
>改进后的用户界面仅适用于沉浸式布局。 移动网站/应用程序尚不支持这些更改，并将在未来版本中对其进行更新。

![](assets/old-ui.png)
_旧用户界面_

![](assets/home-page-new.jpg)
_新的用户界面_

### 主页

主页采用全新设计，改进了侧面板、顶部标题、增强的课程卡和小组件。

![](assets/new-ui-homepage.png)
_新主页_

### 目录页面

目录页面焕然一新，过滤器和增强的课程卡经过组织，可提供更好的用户体验。

![](assets/catalog.jpg)
_目录页面_

### “课程概述”页面

课程概述页面焕然一新，其中包含有关课程的更多详细信息。 此页面可帮助学习者获取所需的所有信息。

![](assets/course-overview.jpg)
_课程概述页面_

### 课程卡

课程卡还具有重新设计的布局，可以更有效地显示详细信息。 改进后的课程卡会突出显示注册所需的相关元数据。 此元数据包括正确的发布日期或截止日期、评级和描述及其作者或提供者。

![](assets/old-course-cards.png)
_旧课程卡_

![](assets/new-course-card.jpg)
_新课程卡_

对于从 **LinkedIn** 和 **Go1** 平台导入的课程，课程卡片将显示 LinkedIn **和** Go1 **的**&#x200B;原始发布日期。您还可以在用户界面上查看这些特定的发布日期。

### 侧边栏和搜索栏

侧边栏使用新的 UI 元素进行了更新，看起来更简洁。 新的搜索栏没有搜索按钮，使其外观更加简洁。 学习者可以键入关键字，然后按 Enter 键开始搜索或选择搜索栏下方的结果。

![](assets/side-bar.png)
_侧边栏和搜索栏_

### 刊头 {#masthead}

展示带有嵌入式 URL 的视频或图片轮播。 管理员可以 [上传任何图像或视频](../../administrators/feature-summary/announcements.md#masthead) 资源作为刊头，并为其设置对一组学习者的可见性。

*查看刊头*

*查看我的学习列表*

您还可以向左和向右轻扫以在列表中导航。

要恢复课程，请单击卡片上的“继续&#x200B;]**”**[!UICONTROL ，播放器将启动。

管理员通过管理员应用程序启用/禁用每张培训卡上的图标外观（**设置** > **常规** > **启用培训卡图标**）。

**添加到“我的学习列表”**

将鼠标悬停在&#x200B;**根据您的兴趣领域推荐**&#x200B;和&#x200B;**根据同事的活动列表推荐**&#x200B;中的任何课程卡上，您将可看到将课程添加到&#x200B;**我的学习列表**&#x200B;的选项。 单击 **[!UICONTROL 课程卡片上的 +]** ，课程就会添加到“ **我的学习列表**”中。

![](assets/add-my-learning.png)

*添加到“我的学习列表”*

## 选择技能级别 {#chooseskilllevels}

作为学习者，您可以根据以下级别筛选课程目录：

* 初学者
* 中等
* 高级

选择一个选项，然后您可以根据选择项查看相应课程目录。

![](assets/skill-levels.png)

*选择技能级别*

## 合规性仪表板小组件

合规性信息板小组件允许学习者使用合规性标签筛选截止日期即将到来的课程/学习路径/认证。 此功能适用于所有学习者应用程序，包括 ALM 团队应用程序、AEM、移动应用程序、沉浸式和 SF 应用程序。

![](assets/compliance-status-learner.png)
_合规性仪表板小组件_

## 日历 {#calendar}

显示您的课程和培训排程。 浏览日历以查看后续几个月的培训。

![](assets/learner-calendar.png)

*查看已安排会话的日历*

“日历”小组件具有以下功能。 可查看的内容：

* 按月开展培训。 向左或向右滚动。
* 您可注册的后续教室培训或虚拟教室培训。
* 您已注册的后续教室培训或虚拟教室培训。
* 得到经理批准的教室培训或虚拟教室培训。

![](assets/social-feed.png)

此小组件汇总了某个时段的活动， 其功能包括：

* 显示最近两周发布的帖子。

## 用户资料技能 {#profileskills}

用户资料技能可用于课程推荐。 管理员向某一用户或一组用户分配技能时，系统会将获得的技能添加至学员的用户资料技能中。 如果学员已在自己的用户资料中添加技能，则系统会将此技能包含的所有级别添加至学习者的用户资料技能中。 将鼠标悬停在技能上方时，学员可以看到技能的名称、技能来源、级别、技能完成率和积分。

![](assets/profile-skills.png)
*查看个人资料技能*

在学员注册参加此课程后，系统只会将按分数划分的外部技能添加至用户资料技能中。 此外，学习者还可以搜索、选择外部技能并将其添加到他/她的个人资料中。如果学习者是首次登录学习者应用程序，并且学习者的技能已经具备，则技能会显示在“我的配置文件”中。

## 根据您的兴趣领域推荐 {#recommendationbasedonyourareaofinterest}

根据您选择的兴趣领域显示培训。 该建议由机器学习算法驱动。

![](assets/learner-recommendation.png)

*查看推荐课程*

有关更有针对性的建议，您可以通过单击“查看/更新&#x200B;**”**&#x200B;来更新您的技能。

添加技能后，系统日后将根据您的偏好作出推荐，因此针对性更强、重点更突出。

如果管理员禁用了&#x200B;**探索技能**&#x200B;选项，则您可将感兴趣的领域添加至您的技能中。

推荐课程以卡片形式显示。 将鼠标悬停在卡片上时，您可以看到课程的更多详细信息。

同样支持产品术语。

**业界公认的技能**

如果管理员在管理应用程序中启用了&#x200B;**业界公认**&#x200B;选项，您便能看到技能网络图表。

在管理员将培训类型设置为“业界公认”后，学员才能查看这些技能。

在“技能图”可视化中，您可以搜索并添加单项或多项技能。

![](assets/learner-add-industry-skills.png)

*技能图可视化*

如果要显示帐户中的所有技能，请启用“显示我的帐户&#x200B;**中存在培训的技能”选项**。

添加技能后，您便可看到一幅力导向图，其中所选技能为主顶点，关联技能则为较小顶点。

您选择的技能也会显示在&#x200B;**所选技能**&#x200B;部分。

![](assets/learner-add-industry-skills-1.png)

*选定的技能*

如要添加技能，请单击&#x200B;**[!UICONTROL 添加]**。

## 根据同事的活动推荐 {#recommendationbasedonpeeractivity}

根据同龄人参加的内容显示培训内容。 这同样也由机器学习算法驱动。 这些建议基于对定制和行业相关学习者的培训。
