# Adobe Learning Manager中的一键式注册

通过电子邮件、聊天或其他消息传递平台从管理员处收到深层链接，以注册培训模块或课程。 选择该链接，即可立即注册并启动模块，无需执行其他步骤。

## 注册学习者并将学习者重定向到课程

请注意以下情况及其相应的操作。 作为学习者，如果您单击发送给您的深层链接，根据您所处的场景，会发生以下情况之一：

| SN | 情景 | 操作 |
| --- | --- | --- |
| 1 | 单模块课程，无序 | 此时将自动注册播放器，播放器将会打开。 必须按“播放”才能播放视频。 |
| 2 | 多模块课程和无序 | 您将自动注册并登陆到深层链接所链接的特定模块。 必须按“播放”才能播放视频。 |
| 3 | 多模块课程和订购 | 您将登录到“概述”页面，并显示一条消息，说明您必须先完成前一门课程，然后才能开始当前一门课程，以防您尚未完成前一门课程。 在这种情况下，您将自动注册并登陆到该课程的第一个模块。 |
| 4 | 课程有先决条件 | 如果前提条件均已完成，则可以在播放器中启动模块。 按“播放”播放视频。 如果未完成预修课程，则会显示以下消息：“无法跳过预修课程。 请先完成所有预修课程，然后再开始本课程。 |
| 5 | 有轮候表的课程 | 您将要注册。 您将进入课程概述页面，并且会添加到轮候表中。 您还会看到一则表明课程已轮候的消息。 |
| 6 | 如果您已经注册了为其生成深层链接的课程。 | 您将登陆模块，然后播放器将启动。 按“播放”播放视频。 |

## 身份验证状态和深层链接行为

除了上述各种情况之外，还有另外两个用户行为与身份验证（登录和注销）状态相关。

* 如果您单击深层链接，并且您未登录到Adobe Learning Manager，则您将被重定向到登录页面。 登录后，深层链接将重定向到特定模块。

* 如果您单击深层链接，并且已登录到Adobe Learning Manager，则将直接重定向到该模块。

## 沉浸式支持移动和Web

以下情况适用于Adobe Learning Manager的移动版本和Web版本。

* 如果您没有Adobe Learning Manager的桌面版本，并且收到了深层链接，那么如果您没有登录到Web或移动版本，则选择该链接会将您重定向到浏览器中Adobe Learning Manager的登录页面。
* 如果您没有Adobe Learning Manager的桌面版，并且收到了深层链接，那么如果您已登录到Web或移动版，则选择该链接后，将直接重定向到浏览器中的Adobe Learning Manager模块。

## 浏览器限制的自动播放

单击深层链接时，如果您未登录，该链接会将您重定向到模块或登录页面。 如果模块为视频，则播放器不会自动播放该视频。 在所有情况下，浏览器都会阻止播放器自动播放视频。 必须手动按“播放”才能播放视频。
