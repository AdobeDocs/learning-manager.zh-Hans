# 简介

虚拟教练器是Adobe Learning Manager中的一种变革性功能，旨在弥合理论知识与实际应用之间的关键差距。 不要只依赖被动的学习方法、视频、测验和阅读材料。 虚拟教练为学习者提供互动的沉浸式环境，用于练习高风险的对话和学习软技能。

把虚拟教练想象成一个“全天候练习伙伴”和“沟通飞行模拟器”。 就像飞行员在操作真实飞机之前在飞行模拟器上记录数千小时一样，员工可以在遇到与真实客户、客户或团队成员的类似情况之前，练习与人工智能驱动的人物进行复杂的对话。 Virtual Coach会实时适应学习者的输入，针对他们所说的内容（知识）和所说的内容（交付方式和风格）提供即时、数据驱动的反馈。
