+++更新89：AdobeLearning Manager 2023年3月版

**发行日期：** 2023年4月1日

### 本次更新中的新增功能

**对讲师指导培训(ILT)体验的增强**

对讲师指导培训(ILT)体验进行了一些改进。 主要增强功能包括：根据位置筛选教室授课的能力、在不损失进度的情况下切换实例(VILT)的能力、用于管理预订讲师和课堂中冲突的新“日程安排助理”、将“技能”附加给讲师并根据技能选择讲师的能力。

**观察核对表的改进**：

现在，作者可以选择“Manager”和“Store Manager”作为检查清单的观察者。 经理无需将角色切换为讲师，即可在Manager界面中查看和完成检查单。 向经理分配清单后，经理会收到通知。

**使用任意应用程序/智能手机摄像头扫描Learning Manager二维码**

学习者现在可以使用任何二维码扫描应用程序或其智能手机摄像头扫描Learning Manager生成的课程注册、完成等二维码。

**报告增强功能**

新的“讲师利用率”报告、“培训重新访问”报告、“工作辅助”报告和其他报告增强功能。

**支持“混合”会话**

AdobeLearning Manager现在支持创建“混合”讲师指导培训(ILT)会话。 可以创建包含可选位置信息的虚拟ILT会话，以便学习者可以亲自参加会话（如果在该位置可用）。

**更好地跟踪教室和虚拟ILT的进度**

教室和虚拟ILT模块现在可以报告学习者的测验状态（通过或失败）以及出勤状态。 因此，可考虑出席情况以及测试是否成功来决定学习者的进度。

**Microsoft TeamsAdobeLearning Manager应用程序**

Microsoft Teams上新的AdobeLearning Manager应用程序旨在促进工作流程中的学习并提升社交学习。 学习者无需切换到Microsoft Teams即可访问浏览器平台内的学习内容。 请联系CSAM以获取MS Teams上AdobeLearning Manager应用程序的Beta版。

### 本次更新中修复的错误

**课程**

* 课程处于“UNDER_CONSTRUCTION”状态时，自定义作者无法预览模块。 响应显示错误404。
* 如果课程标题超过特定字符限制，作者应用程序的课程/添加页面上的课程标题无法完整显示。

**作者**

* 在作者应用程序中，创建课程时，课程的标题（如果课程很长）超出了页面边界。
* 有时，即使未选择作者，系统也会添加课程。

**信息板报告**

* 当界面语言为英语时，工具提示显示良好，但当界面语言不同时，会引发控制台错误。
* 在学习者信息板中，将“必填”重命名为“必填”。

**讲师应用程序**

* 讲师应用程序中的时间格式与其他应用程序不一致。

**Social**

* 对于某些类型的帖子，发布后，社交讨论区无法按预期打开。

**管理员**

* 预览课程时，具有自定义角色的用户无法下载资源。

**电子邮件模板**

* 当学习者取消注册包含课堂/虚拟教室课程的学习计划时，不会收到任何取消电子邮件。

**工作辅助**

* 无法在“工作辅助”小组件上看到课程名称。

**发布**

* 在ALM中发布模块后，在Adobe Captivate中添加的模块描述在Learning Manager中不可见。

**活动字段**

* 处理包含大量记录的CSV文件时，耗时很长，在此期间，如果用户登录并输入其中一个属性的值，则可能会创建一个新用户组，可能会导致CSV错误。 要解决此问题，请在导入CSV的过程中禁用“活动字段”属性弹出消息，并在CSV上传完成后重新启用该消息。
* 如果用户csv文件中的列与外部用户活动字段的名称相同，则csv上传会失败。

**与API相关的修复**

* 在learningObjects响应中，缺少书签属性。
* 为已删除的用户生成OAUTH刷新令牌时，会创建访问条目。
* 学习对象API返回错误的loFormat，因为计算课程类型和核心内容时考虑到了预习模块。

**本次更新中的已知问题**

* 在Safari浏览器、移动设备和iPad MS Teams应用程序中，学习者目录上的“共享”按钮无法按预期工作。
* 在其他计算机上删除应用程序后，“活动”选项卡中不显示通知。
在iPhone 14上单击应用程序的“活动”选项卡中的通知时，不会发生任何情况。
* 在MS Teams应用程序上，Learning Manager通知（已完成、已注册、截止日期和超时）不会在“活动”选项卡中显示课程状态和名称。
* 当集成管理员未批准MS Teams应用程序时，会显示一个包含XML内容的弹出窗口。
* 语言发生变化时，MS Teams上AdobeLearning Manager应用程序上的用户界面语言有时无法按预期发生变化。
* 当焦点位于Iframe（“主页”和“目录”选项卡）内时，您无法与第一个通知进行交互。

**AdobeLearning Manager移动应用程序的限制**

* 查看脱机内容。
* “目录/我的学习”页面上的“网格”/“列表”视图。
* 多次尝试参加课程。
* 课程卡上的注册截止日期。
* 在iOS设备上，当应用程序在前景中时，不显示推送通知。
* 推送通知中的深层链接不会重定向到预期的登陆页面。

## 新增功能 {#Newfeatures-2}

### 工作辅助

工作辅助是一个培训内容知识库，可供学习者访问，无需任何注册或完成标准。 学习者可参考这些工作辅助，获取在公司内执行任何活动或任务的相关协助。 管理员可以跟踪每个工作辅助的下载次数。

有关此功能的更多信息，请参阅  [工作辅助帮助](../learners/feature-summary/job-aids.md).

### 公告

公告是一种多媒体消息（文本、图像或视频），由管理员起草并向一组特定用户进行广播。 使用公告激励学习者参加培训，从而建立学习文化。

有关此功能的更多信息，请参阅  [公告帮助](../learners/feature-summary/announcements.md).

### Tin Can API支持

AdobeLearning Manager支持Tin Can API（也被称为体验API或xAPI）规格。 您可以上传并跟踪Tin Can API兼容内容，方法与跟踪SCORM和AICC内容相似。

有关更多信息，请联系Adobe支持团队。

### 课程顺序

您可以通过指定后续课程或任何学习活动自动创建学习路径。

有关学习计划的事件已更新。 增加了几项新活动。 请参阅  [学习方案](../learners/feature-summary/learning-programs.md) 了解更多信息。

### 笔记提醒

如果使用课程时记了笔记，Learning Manager会在15天后通过发送通知来提醒您复习笔记。

### 组级别游戏

管理员可以通过更改范围设置来定义游戏的范围。 您可以选择性地在个人资料相似的用户、组或位置之间启用游戏。 请参阅  [游戏](../learners/feature-summary/gamification.md) 功能，以获取更多信息。

### 法语和德语支持

Learning Manager应用程序提供法语和德语版本。 您可以自定义用于反馈、课程实例和通信的语言。
